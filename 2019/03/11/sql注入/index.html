  • 找到注入点且得到闭合符号

    sql语句中 –空格 是注释的意思，在url中+号编码为空格
    即 ‘id=1’– ‘ 正常回显，–空格将最后的单引号注释掉了

以下内容为部分程序代码:
$id=$GET[‘id’]; //接受get传递的参数名为id的值并赋值给变量id
$sql=”SELECT * FROM users WHERE id=’$id’ LIMIT 0,1”; // 把查询到的结果赋值给变量名sql
一对单引号为闭合字符

    order by 3 对
    得一共3列

    爆出显示位为后两位，可以在2，3的位置做手脚
